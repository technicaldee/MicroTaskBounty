# MiniPay Web App

React web application for the MiniPay micro-task bounty platform.

## Setup

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## Features

- React + TypeScript + Vite
- React Router for navigation
- Redux Toolkit for state management
- Ethers.js for blockchain interactions
- Web APIs for camera and geolocation
- Responsive design

## Configuration

Update contract addresses in `src/services/blockchain.ts` after deploying contracts.

## Wallet Connection

The app supports:
- MetaMask (automatic detection)
- Private key (for testing)
